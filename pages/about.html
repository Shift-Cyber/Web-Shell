Web Shell

This product is essentially a fork of pyxtermjs with a management console for instantiating and deploying sessions. The backend is an autoscaling Kubernetes cluster that dynamically deploys via image. Administrators are able to define custom images and sessions are fully interactive over routed WebSockets. This software is open source and developed under the MIT License, please contribute whatever features you feel would be benifical.
